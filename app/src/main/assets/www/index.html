Google Calendar API Setup

To connect your real Google Calendar and sync automatically in the background, follow this quick 2-minute guide:

  1. Open the Google Cloud Console in your browser.
  2. Create a project, search for Google Calendar API, and click Enable.
  3. Configure the OAuth consent screen (choose External, add your email, and add the scope .../auth/calendar.events). Important: Add your Google email address under "Test Users"!
  4. Go to Credentials click Create Credentials -> OAuth client ID:
    • Application Type: Web application
    • Authorized Javascript Origins: http://localhost:8080 (or your current port)
    • Authorized Redirect URIs: http://localhost:8080 (or current redirect port)
  5. Copy your generated **Client ID**, paste it in the header input, and click **Save**!
